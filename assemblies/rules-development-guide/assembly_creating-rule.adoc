= Creating custom rules

:context: creating-rule

[role="_abstract"]
To analyze applications in `Python`, `Node.js`, `Go`, and `C#`, the Architect must create a custom rule. A custom rule includes conditions and patterns that describe a violation. You must resolve violations when refactoring applications before migrating them to a target platform. 

You can see examples that describe how to create a custom rule. This assumes that you already have {ProductShortName} installed. See the {ProductShortName} {CLIName} Guide for installation instructions.
