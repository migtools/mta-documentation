= Configuring the {ProductName} instance environment

:context: configuring-mta-instance-environment

[role="_abstract"]
To adjust migration projects to specific requirements, configure the {ProductFullName} instance environment.

You can configure the {ProductShortName} instance environment by setting up the following parameters: 		

* General	 				
* Credentials					
* Repositories 				
* Proxy
* Custom migration targets			
* Issue management			
* Assessment questionnaires	

== Prerequisites

* You installed the {ProductShortName} UI. For more information, see link:https://docs.redhat.com/en/documentation/migration_toolkit_for_applications/8.0/html/install-guide/installing-mta-ui_installing-mta-title[Installing the {ProductName} user interface].
