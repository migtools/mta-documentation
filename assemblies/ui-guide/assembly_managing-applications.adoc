[role="_abstract"]
Add applications, assign credentials, import app lists, create migration waves, and create Jira issues in the {ProductFullName} UI to prepare for assessment and analysis.

You can use the {ProductFullName} user interface (UI) to perform the following tasks to prepare your applications for the assessment and analysis: 	

* Add applications
* Assign application credentials
* Import a list of applications
* Download a CSV template for importing application lists
* Create application migration waves
* Create Jira issues for migration waves

{ProductShortName} UI applications have the following attributes:

* Name (free text)
* Description (optional, free text)
* Business service (optional, chosen from a list)
* Tags (optional, chosen from a list)
* Owner (optional, chosen from a list)
* Contributors (optional, chosen from a list)
* Source code (a path entered by the user)
* Binary (a path entered by the user)
