= Persistent volume requirements

[role="_abstract"]
The Solution Server component requires a backend database to store code changes from previous analyses. 

If you plan to enable Solution Server, you must create a `5Gi` `RWO` persistent volume used by the {mta-dl-plugin} database. See xref:persistent-volume-requirements[Persistent volume requirements] for more information.
