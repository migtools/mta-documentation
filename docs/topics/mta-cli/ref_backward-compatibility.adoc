= Compatibility with deprecated commands

[role="_abstract"]
To ensure that the restructured commands are compatible with the previous command structure, the {ProductFullName} CLI converts an unsupported command option to the new command.

For example, if you enter:

[subs="+quotes"]
----
*$ mta-cli analyze --list-targets*
----

The {ProductShortName} CLI processes the command as:

[subs="+quotes"]
----
*$ mta-cli rules list-targets*
----
