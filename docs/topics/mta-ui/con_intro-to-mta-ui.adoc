= Introduction to the {ProductName} user interface

[role="_abstract"]
You can manage your application migration projects by using the {ProductFullName} user interface (UI).

You can use the {ProductShortName} UI to assess and analyze applications for risks and suitability for migration to hybrid cloud environments on Red Hat OpenShift. By using the UI, you can perform the following tasks:

* Assess the risks involved in containerizing an application for hybrid cloud environments on Red Hat OpenShift.
* Analyze the changes that you might need to apply to the code of an application to containerize this application.
