= Assigning credentials to an application

[role="_abstract"]
To enable the {ProductFullName} to access secure source repositories, assign credentials to your applications. Associating credentials with your application ensures that the analysis engine can retrieve and process the source code and binaries.

.Prerequisites

* You added or imported your applications to the {ProductFullName} user interface (UI). For more information, see the following procedures:
** xref:adding-applications[Adding a new application to the {ProductShortName} UI]
** xref:importing-an-app-list[Importing a list of applications]

.Procedure

. In the *Migration* view, click *Application inventory*.
. Click the Options menu ({kebab}) to the right of *Analyze* and select *Manage credentials*.
+
// ![](/Tackle2/AddingApps/ManageCred.png)

. Select one credential from the *Source credentials* list and from the *Maven settings* list.
. Click *Save*.
