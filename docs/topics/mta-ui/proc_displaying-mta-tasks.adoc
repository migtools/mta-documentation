= Displaying {ProductShortName} tasks

[role="_abstract"]
To display information related to {ProductFullName} user interface tasks, access *Task Manager*.

.Procedure

* Depending on your scenario, display information about {ProductShortName} tasks either of the following ways:

** To display active tasks, open the *Task Manager* drawer by clicking the notifications button.
** To display all tasks, open the *Task Manager* page in the *Migration* view.
