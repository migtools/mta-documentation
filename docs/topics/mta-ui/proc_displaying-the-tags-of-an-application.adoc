= Displaying application tags

[role="_abstract"] 
To verify the application classification, display application tags in the {ProductFullName} user interface (UI).

NOTE: You can display the tags that were attached automatically only _after_ you ran an application analysis.

.Procedure

. In the *Migration* view, click *Application inventory*.
. Click the name of the required application. A side pane opens. 
. Click the *Tags* tab. The tags attached to the application are displayed.
