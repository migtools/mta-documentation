= New features and enhancements

[role="_abstract"]
Review new features and enhancements in {ProductFullName} 8.0.1.


{ProductShortName} supports pre-generated Maven index to reduce binary application analysis latency::
Before this update, the analysis performance for Java binaries degraded because {ProductShortName} relied on Maven search to analyze `Java` binary applications. With this enhancement, {ProductShortName} has pre-generated bundled Maven index that does not require connecting to an external index through a network connection. As a result, {ProductShortName} performs fast, low-memory look-ups of Maven artifacts by using local Maven Search and you can analyze Java binaries in disconnected environments.
