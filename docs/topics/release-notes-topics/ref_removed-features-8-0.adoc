= Removed features

[role="_abstract"]
Review all features removed in {ProductFullName} 8.0.0. All removed features were deprecated in earlier releases and are no longer supported. 

{ProductShortName} removes support for XML rules::
In {ProductFullName} {ProductVersion}, {ProductShortName} removes XML rules and no longer supports them in the {ProductShortName} user interface (UI) and command-line interface (CLI).
+
(link:https://issues.redhat.com/browse/MTA-5357[MTA-5357])


{ProductShortName} removes support for the Eclipse IDE plugin::
{ProductShortName} removes the Eclipse Integrated Development Environment (IDE) plugin for {ProductFullName} to analyze the effort for migrating and modernizing your applications and no longer supports it. As an alternative, you can use the {ProductName} extension for Visual Studio Code.
