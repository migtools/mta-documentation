= Provider capabilities in custom rules

[role="_abstract"]
As an Architect, you define conditions and patterns in custom rules that {ProductShortName} uses to match against source code and dependencies in an analysis. You can specify the conditions in the `when` block of the rule. Each provider offers capabilities to match rule patterns.

The `when` block contains a single condition, but that condition might have multiple conditions nested under it.

----
when:
  <condition>
    <nested-condition>
----
