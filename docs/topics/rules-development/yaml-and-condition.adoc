= AND condition

[role="_abstract"]
The `and` condition performs a logical AND operation on the results of an array of conditions. The condition matches when _all_ of its child conditions match, for example:

[source,yaml, subs="+quotes"]
----
when:
  and:
    - _<condition1>_
    - _<condition2>_
----

For an example rule condition, see:

[source,yaml]
----
when:
  and:
    - java.dependency:
        name: junit.junit
        upperbound: 4.12.2
        lowerbound: 4.4.0
    - java.referenced:
        location: IMPORT
        pattern: junit.junit
----
