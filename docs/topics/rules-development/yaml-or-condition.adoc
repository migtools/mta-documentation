= OR condition

[role="_abstract"]
The `or` condition performs a logical OR operation on the results of an array of conditions. The condition matches when _any_ of its child conditions match.

[source,yaml, subs="+quotes"]
----
when:
  or:
    - _<condition1>_
    - _<condition2>_
----

For example:

[source,yaml]
----
when:
  or:
  - java.dependency:
      name: junit.junit
      upperbound: 4.12.2
      lowerbound: 4.4.0
  - java.referenced:
      location: IMPORT
      pattern: junit.junit
----
